결제까지 마친 유저가 가장 큰 가치를 느끼는 순간이자, 이 서비스의 핵심 엔진인 'AI 지원서 작성 에디터' 화면이군요!

지원사업(초창패, 예창패, 소상공인 지원금 등)에 도전하는 대표님들이 가장 두려워하는 것은 '빈 한글(HWP) 화면'을 마주하는 것입니다. 따라서 이 화면의 핵심 디자인 목표는 **"유저가 글을 쓴다는 느낌을 주지 않고, AI와 대화하다 보니 어느새 전문적인 사업계획서가 완성되어 있게 만드는 것"**입니다.

피그마 디자이너에게 전달할 수 있도록, 화면을 좌우로 나누는 스플릿 뷰(Split View) 구조 형태의 UI/UX 기획안을 정리해 드립니다.

🎨 피그마 디자이너 전달용: AI 지원서 작성 에디터 구조
1. 전체 레이아웃: 스플릿 뷰 (좌: 대화형 입력창 / 우: 실시간 문서 프리뷰)
화면을 세로로 4:6 또는 3:7 비율로 나누어, 왼쪽에서는 가볍게 정보를 입력하고 오른쪽에서는 정부 양식에 맞춰 실시간으로 글이 써지는 것을 시각적으로 보여줍니다.

2. 좌측 패널 (AI 인터뷰어 & 가이드 영역)
유저가 긴 글을 쓸 필요 없이, 단답형이나 키워드만 던져주면 되도록 설계합니다.

진척도 바 (Progress Bar): 현재 지원서의 어느 항목(예: 문제인식 -> 해결방안 -> 성장전략 -> 팀 구성)을 작성 중인지 보여줍니다.

AI의 단계별 질문 (채팅 또는 폼 형태):

예: "대표님의 핵심 아이템과 타겟 고객을 짧게 적어주세요."

키워드 입력창 & 추천 태그: * 유저는 문장이 아닌 단어만 입력해도 됩니다.

(핵심 디테일) 예를 들어, 유저가 입력창에 "수제 마카롱, 온라인 주문 제작, 20대 여성" 정도로만 키워드를 툭툭 던져 넣을 수 있도록 가벼운 UI를 제공합니다.

'AI 초안 생성' 버튼: 키워드를 다 넣고 누르면 마법이 시작되는 버튼입니다. 화려한 그라데이션이나 애니메이션을 넣어 기대감을 줍니다.

3. 우측 패널 (실시간 문서 완성 영역)
좌측에서 입력한 키워드를 바탕으로, 정부기관에서 요구하는 딱딱하고 전문적인 문체로 변환된 텍스트가 채워지는 공간입니다.

문서 양식 뷰어: 실제 공고의 HWP 양식(표, 텍스트 박스 등)을 웹 화면에 깔끔하게 구현해 둡니다.

타이핑 애니메이션: 좌측에서 버튼을 누르면, 우측 문서 빈칸에 AI가 글을 실시간으로 타이핑해 나가는 애니메이션을 적용해 주세요. (유저에게 'AI가 내 일을 대신해주고 있다'는 효능감을 줍니다.)

변환 예시 시각화: (좌측) "수제 마카롱, 온라인 주문 제작" ➡️ (우측) "본 사업은 MZ세대 소비 트렌드를 반영한 프리미엄 수제 디저트(마카롱)의 온라인 맞춤형 주문/결제 시스템을 구축하여..."

부분 수정 툴팁 (마이크로 인터랙션): AI가 써준 문장 중 특정 블록에 마우스를 올리면 [더 길게], [더 전문적으로], [수치 강조], [직접 수정] 같은 둥근 팝업 메뉴가 뜨도록 디자인해 주세요.

4. 상하단 고정 영역 (GNB & Floating Bar)
상단 헤더: "2026년 소상공인 스마트상점 기술보급사업 지원서 작성 중" (현재 공고명 명시) / 자동 저장 시간 표시

하단 액션 바:

[임시저장]

[전문가 첨삭 요청하기] (프리미엄 옵션)

[HWP로 다운로드], [PDF로 다운로드]

💡 UX/UI 기획 꿀팁 (디자이너 강조 사항)
어려운 용어 번역기: 정부지원금 특유의 어려운 단어(예: 정량적 목표, BM, 스케일업 등) 옆에 (?) 아이콘을 배치하고, 누르면 AI가 아주 쉬운 일상어로 설명해 주는 툴팁을 꼭 넣어주세요.

빈칸 공포증 없애기: 처음 에디터에 진입했을 때, 우측 문서 영역이 텅 비어있지 않고 흐릿한 회색 글씨(Placeholder)로 "이곳에 AI가 정부 부처 스타일의 맞춤형 사업계획서를 작성합니다"라는 텍스트를 채워두는 것이 좋습니다.

이렇게 화면이 구성되면 유저는 "키워드 몇 개 넣었을 뿐인데 전문적인 지원서가 뚝딱 나오네? 수수료 낸 보람이 있다!"라고 확실하게 체감할 수 있습니다.
